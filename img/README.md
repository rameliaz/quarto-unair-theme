# Logo Placeholder

This folder should contain your Universitas Airlangga logo files:

## Required Files

1. **logo.png** - Regular UNAIR logo for light backgrounds
   - Use official horizontal logo (brandmark + wordmark + tagline)
   - Format: PNG with transparent background
   - Recommended size: 300-400px wide
   - This appears on: regular slides, title slide

2. **logo_white.png** - White version for dark backgrounds  
   - Same design as logo.png but all white
   - Format: PNG with transparent background
   - This appears on: section divider slides with blue background

## Where to Get Official Logos

Contact Universitas Airlangga's Communications/Marketing department or visit the official brand guidelines repository.

## Logo Specifications (per UNAIR Brand Guidelines 2025)

- Minimum size: 15mm diameter for brandmark
- Clear space: 1/3 of logo height on all sides
- Preferred position: Top-right corner with 15px margin
- Do not modify, rotate, or distort the logo
- Do not change colors (except using official white version)

## For Testing

If you don't have official logos yet, you can create simple placeholders:
- Use any rectangular PNG image
- Name them logo.png and logo_white.png
- Place them in this folder
